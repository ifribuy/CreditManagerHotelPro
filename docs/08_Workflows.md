# Business Workflows

## Purpose

Define end-to-end operational workflows for CreditManagerHotelPro.

------------------------------------------------------------------------

# 1. Excel Import Workflow

User → Select File → Validate Columns → Parse Data → Save Import History
→ Calculate KPIs → Refresh Dashboards → Audit Log

------------------------------------------------------------------------

# 2. Daily Collection Workflow

Dashboard → Critical Accounts → Prioritize Customers → Send Reminder →
Phone Call → Register Notes → Promise to Pay → Follow-up → Close Case

------------------------------------------------------------------------

# 3. Credit Review Workflow

Customer Profile → Analyze Exposure → Review Payment History → Review
Risk Score → Decision: - Approve Credit - Reduce Limit - Increase
Limit - Block Credit

Record Decision → Notify Stakeholders → Audit

------------------------------------------------------------------------

# 4. Credit Block Workflow

Risk Threshold Reached → Manager Review → Approve Block → Customer
Status = BLOCKED → Notify Sales & Finance → Prevent New Credit Sales

------------------------------------------------------------------------

# 5. Credit Unblock Workflow

Payment Received → Review Outstanding Balance → Manager Approval →
Status = ACTIVE → Audit → Notification

------------------------------------------------------------------------

# 6. Litigation Workflow

Create Litigation → Attach Documents → Assign Owner → Track Progress →
Settlement / Legal Action → Close Litigation

------------------------------------------------------------------------

# 7. Promise to Pay Workflow

Create Promise → Due Date → Reminder Before Due → Paid = Close Not Paid
= Escalate

------------------------------------------------------------------------

# 8. Month-End Workflow

Import Final Balance → Recalculate KPIs → Compare Previous Month →
Generate Executive Report → Generate Credit Manager Report → Export PDF
& Excel → Archive Snapshot

------------------------------------------------------------------------

# 9. AI Recommendation Workflow

Analyze Portfolio → Detect Risks → Rank Priorities → Recommend Calls →
Recommend Blocks → Forecast Cash → Generate Summary

------------------------------------------------------------------------

# 10. Executive Reporting Workflow

Select Period → Generate Charts → Generate KPIs → Generate Narrative →
Export PDF → Email / Share

------------------------------------------------------------------------

# Workflow Statuses

Customer: ACTIVE WARNING RESTRICTED BLOCKED LITIGATION CLOSED

Reminder: PLANNED SENT ANSWERED COMPLETED ESCALATED

Promise: ACTIVE DUE FULFILLED OVERDUE CANCELLED

Litigation: OPEN UNDER REVIEW NEGOTIATION RESOLVED CLOSED

Import: VALIDATING IMPORTING COMPLETED FAILED ARCHIVED
