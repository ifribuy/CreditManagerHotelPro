# Design System

## Vision

Create a premium, enterprise-grade visual language consistent across the
entire application.

## Design Principles

-   Data-first
-   Elegant simplicity
-   High information density
-   Consistent spacing
-   Smooth animations
-   Accessible

## Color Palette

Primary Background: #0F172A Surface: #1E293B Secondary Surface: #334155

Primary Blue: #2563EB Success Green: #10B981 Warning Orange: #F59E0B
Critical Red: #EF4444 AI Purple: #8B5CF6

Primary Text: #FFFFFF Secondary Text: #CBD5E1 Muted Text: #94A3B8

## Typography

Headings: - Poppins SemiBold

Body: - Inter

Numeric KPIs: - JetBrains Mono

## Grid & Spacing

Base unit: 8 px

Spacing: - XS: 4 - SM: 8 - MD: 16 - LG: 24 - XL: 32 - XXL: 48

Card Radius: 16--20 px

## Components

### KPI Card

Contains: - Icon - Title - Value - Trend % - Sparkline - Status color

### Buttons

Primary Secondary Success Warning Danger Outlined Text

### Tables

-   Sticky header
-   Zebra rows
-   Column sorting
-   Filtering
-   Search
-   Export

### Forms

-   Filled inputs
-   Date picker
-   Currency input
-   Search field
-   Multi-select

### Navigation

Bottom Navigation: - Dashboard - Clients - Analytics - Reports - More

Side Drawer: - Administration - Settings - Help

## Icons

Outlined style

Categories: - Finance - Credit - Collection - Reports - AI - Settings -
Alerts

## Charts Style

-   Rounded corners
-   Soft shadows
-   Smooth animation
-   Interactive tooltips
-   Export support

## Animations

-   KPI count-up
-   Card fade-in
-   Chart draw animation
-   Skeleton loading
-   Progress indicators

Duration: 150--350 ms

## Themes

Dark Mode: Default

Light Mode: Optional

## Accessibility

-   WCAG compliant contrast
-   Large touch targets
-   Dynamic font scaling
-   Screen reader labels
