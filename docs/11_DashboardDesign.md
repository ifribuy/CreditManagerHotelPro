# Dashboard Design Specification

## Objective

Design premium analytics dashboards inspired by Microsoft Power BI,
Oracle Fusion Cloud and Stripe Dashboard.

## Global Design

Theme: - Dark mode by default - Light mode available

Colors: - Background: #0F172A - Surface: #1E293B - Secondary: #334155 -
Primary: #2563EB - Success: #10B981 - Warning: #F59E0B - Critical:
#EF4444 - AI: #8B5CF6

Typography: - Headings: Poppins SemiBold - Body: Inter - Numbers:
JetBrains Mono

## Dashboard Layout

Header - Hotel selector - Period selector - Notifications - AI
Assistant - User profile

Row 1 - Total Receivables - DSO - Recovery Rate - Cash Forecast

Row 2 - Aging Distribution - Portfolio Trend - Risk Gauge

Row 3 - Top 10 Debtors - Critical Accounts - Payment Promises

Row 4 - AI Recommendations - Alerts - Quick Actions
