# Functional Specifications

## 1. Application Goal

Provide a complete Android platform for Hotel Credit Management from
Excel aging balance import to executive reporting.

## 2. Main Navigation

-   Dashboard
-   Clients
-   Analytics
-   Collections
-   Credit
-   Reports
-   Notifications
-   Administration

## 3. Dashboard

Features: - KPI Cards - Trend indicators - Top 10 Debtors - Cash
Forecast - Risk Gauge - Alerts - AI Recommendations - Monthly
Comparison - Yearly Comparison

## 4. Excel Import

-   Select file
-   Validate structure
-   Preview
-   Import
-   Save history
-   Compare with previous import
-   Error report

## 5. Client Module

Client profile includes: - Aging balance - Risk score - Credit limit -
Payment history - Collection history - Documents - Notes - Promises to
pay - Litigation

Actions: - Block credit - Unblock credit - Create reminder - Register
payment - Add note - Export client report

## 6. Analytics

-   Portfolio evolution
-   Aging evolution
-   DSO
-   ADD
-   Recovery Rate
-   CEI
-   Exposure
-   Risk distribution
-   Monthly trends
-   Yearly trends

## 7. Charts

-   Line
-   Area
-   Donut
-   Bar
-   Stacked Bar
-   Heat Map
-   Treemap
-   Gauge
-   Waterfall
-   Sparkline

## 8. Reports

Generate: - Executive Report - Credit Manager Report - Collection
Report - Risk Report - DSO Report - Cash Forecast Report

Formats: - PDF - Excel - CSV

## 9. Notifications

-   Critical customer
-   Credit limit exceeded
-   New litigation
-   Promise due
-   Import completed
-   Monthly report ready

## 10. Administration

-   Hotels
-   Companies
-   Users
-   Roles
-   Permissions
-   KPI settings
-   Themes
-   Report templates
-   AI settings

## 11. Non-functional Requirements

-   Fast import
-   Offline support
-   Dark/Light mode
-   Responsive layouts
-   Secure authentication
-   Audit trail
-   Modular architecture
