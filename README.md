# CreditManagerHotelPro

## README

> Master specification entry point for the Hotel Credit Management
> platform.

## Vision

CreditManagerHotelPro is a professional Android application dedicated to
Hotel Credit Management. Its purpose is to become the reference platform
for managing hotel receivables, credit limits, collections, analytics,
executive reporting and AI-assisted decision making.

The project is designed as a commercial-grade product rather than a
simple mobile application.

------------------------------------------------------------------------

# Objectives

-   Import Aging Balance Excel files.
-   Monitor customer receivables.
-   Automate KPI calculations.
-   Produce executive dashboards.
-   Generate PDF and Excel reports.
-   Manage credit limits.
-   Manage collection workflows.
-   Compare monthly and yearly periods.
-   Forecast collections and cash flow.
-   Provide AI recommendations.
-   Support multiple hotels and companies.
